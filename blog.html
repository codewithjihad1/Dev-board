Question-1: What are the different ways to select an element in the DOM?

আমরা Javascript এর মাধ্যমে DOM এর element গুলোকে সিলেক্ট করার জন্য কিছু মেথড ব্যবহার করে থাকি মেথড গুলো হলো ১. getElementById(): ID ব্যবহার করে DOM element সিলেক্ট করা হয়। ২. getElementsByClassName(): Class name ব্যবহার করে DOM element সিলেক্ট করা হয়। ৩. getElementsByTagName(): Tag name ব্যবহার করে DOM element সিলেক্ট করা হয়। ৪. querySelector(): CSS selector ব্যবহার করে DOM element সিলেক্ট করা হয়। ৫. querySelectorAll(): CSS selector ব্যবহার করা হয় এই মেথড এর মাধ্যমে একসাথে একাধিক Element select করা যায়। এবং এটি একটি NodeList Return করে থাকে।

Question-2: What is the difference between innerHTML, innerText, and textContent?

  1. innerHTML
    • এটি HTML কোড সহ যেকোনো কন্টেন্ট সেট বা রিটার্ন করতে পারে।
    • HTML ট্যাগ এবং স্ট্রাকচার সমর্থন করে, মানে HTML পার্স করতে পারে।
    • তবে, এটি XSS (Cross-Site Scripting) আক্রমণের ঝুঁকি বাড়িয়ে দেয়, যদি আপনি ইউজারের ইনপুট সরাসরি এতে যুক্ত করেন।
  2. innerText
    • এটি শুধুমাত্র ভিজিবল (দৃশ্যমান) টেক্সট ফেরত দেয় বা সেট করে।
    • HTML ট্যাগকে ইগনোর করে, শুধুমাত্র সাধারণ টেক্সট দেখায়।
    • হিডেন (display: none) এলিমেন্টের টেক্সট রিটার্ন করবে না।
    • এটি ধীরগতির হতে পারে, কারণ এটি CSS স্টাইল ক্যালকুলেশন করে।
  3. textContent
    • এটি এলিমেন্টের সম্পূর্ণ টেক্সট (হিডেন এলিমেন্টসহ) রিটার্ন বা সেট করতে পারে।
    • HTML ট্যাগ ইগনোর করে, কিন্তু হিডেন এলিমেন্টের টেক্সট ধরে রাখে।
    • এটি সবচেয়ে পারফরম্যান্স-ফ্রেন্ডলি কারণ এটি স্টাইল ক্যালকুলেশন করে না।

Question-3: What is event delegation in the DOM?

Event Delegation হল এক ধরনের JavaScript ইভেন্ট হ্যান্ডলিং টেকনিক, যেখানে কোনো parent element-এর মাধ্যমে তার child elements-এর ইভেন্ট হ্যান্ডল করা হয়। অর্থাৎ, আলাদা আলাদা ইলিমেন্টে ইভেন্ট লিসেনার যোগ করার পরিবর্তে, একটি parent element-এ ইভেন্ট লিসেনার সেট করে সবগুলো child element-এর ইভেন্ট ম্যানেজ করা যায়।

Question-4: What is event bubbling in the DOM?

Event Bubbling হল DOM ইভেন্ট প্রোপাগেশনের (Propagation) একটি কনসেপ্ট, যেখানে কোনো child element-এ ইভেন্ট ট্রিগার হলে, সেই ইভেন্ট parent element-এর দিকেও ছড়িয়ে যায় (bubble করে)। অর্থাৎ, যদি আপনি কোনো inner element-এ ক্লিক করেন, তাহলে সেই ইভেন্ট প্রথমে সেই ইলিমেন্টেই ট্রিগার হবে, তারপর এর প্যারেন্ট, তারপর তার প্যারেন্ট... এভাবে একেবারে Document (HTML Root) পর্যন্ত পৌঁছাতে পারে।

Question-5: How do you create, add, and remove elements using JavaScript?

JavaScript ব্যবহার করে DOM (Document Object Model) এর মধ্যে নতুন এলিমেন্ট তৈরি, সেটি যোগ (append) এবং প্রয়োজনে মুছতে (remove) পারা যায়। এখানে আমরা তিনটি কাজ শিখব:
1️⃣ নতুন এলিমেন্ট তৈরি করা
2️⃣ DOM-এ এলিমেন্ট যোগ করা
3️⃣ DOM থেকে এলিমেন্ট মুছে ফেলা
১. নতুন এলিমেন্ট তৈরি করা
DOM-এ নতুন এলিমেন্ট তৈরি করতে document.createElement() ব্যবহার করা হয়।
২. DOM-এ এলিমেন্ট যোগ করা
নতুন তৈরি করা এলিমেন্টকে DOM-এ যোগ করার জন্য appendChild() ব্যবহার করা হয়।
৩. DOM থেকে এলিমেন্ট মুছা
DOM থেকে কোনো এলিমেন্ট মুছে ফেলার জন্য remove() method ব্যবহার করা হয়।
